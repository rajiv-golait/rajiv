# Project Context: Falcon Portfolio

## Objective
Build a production Next.js 15 portfolio for Rajiv Golait, reflecting his dual identity as an E&TC Engineering Student / AI builder and a Visual Storyteller (shot_by_f.a.l.c.o.n).

## Tech Stack
- Next.js 15 (App Router, TS)
- Tailwind CSS v4
- Sanity.io CMS
- Framer Motion

## Design Identity
Spider-Verse cinematic aesthetic. Pure comic-book aesthetics with heavy halftones, async borders, neon glows, and hard 3px drop shadows.
